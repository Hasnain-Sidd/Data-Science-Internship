Task 2:🔍 Credit Risk Prediction Dashboard | Machine Learning Project
I developed a Credit Risk Prediction system using a real-world financial dataset to classify whether a loan applicant is likely to default (high risk) or not (low risk). This project involved comprehensive data preprocessing, including feature scaling using StandardScaler, and the implementation of four powerful classification algorithms:
✅ Logistic Regression
✅ K-Nearest Neighbors (KNN)
✅ Random Forest
✅ Decision Tree
Among these, the Decision Tree classifier delivered the best accuracy and was selected as the final model. The project also includes detailed confusion matrix analysis for all models, offering a clear comparison of performance across metrics like precision, recall, and F1-score.
To enhance usability, I built a clean and interactive dashboard that allows users to visualize key insights and predictions. This dashboard can assist financial institutions in assessing loan risks quickly and effectively.
